Live Job Market Analytics Dashboard

Overview

This project analyzes live Data Analyst job postings in India using Python and Power BI. It provides insights into hiring trends, in-demand skills, and geographic distribution of opportunities.

Features

Dashboard 1: Market Overview

* Total Jobs Posted
* Total Hiring Companies
* Total Locations
* Latest Job Posted Date
* Top Hiring Companies
* Top Hiring Locations

Dashboard 2: Skills Demand Analysis

* Top In-Demand Skills
* Skill Frequency Analysis
* Skills Distribution

Dashboard 3: Geographic Insights

* Geographic Distribution of Jobs
* Top Hiring Cities
* Regional Hiring Analysis

Tech Stack

* Python
* Pandas
* Requests
* Power BI
* Adzuna API

How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Fetch latest jobs:
   python Scripts/fetch_jobs.py

3. Extract skills:
   python Scripts/extract_skills.py

4. Open Dashboard/Dashboard.pbix

5. Click Refresh in Power BI.
